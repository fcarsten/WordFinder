Word Finder vX.X

You can access the Settings screen from the cog icon in the APP Bar:

You can change the following settings:

Select which dictionary to use to check words and solve the board

Select the scoring mode. Options are Word Count or Word Value

Select words with only 3 letters are allowed

Select whether prefixal words are automatically added to your result list when entering the longer word. If enabled, entering e.g. "cats" would also automatically add 'cat' to the results list
